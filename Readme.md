Flow.Launcher.Plugin.CdList
==================

A plugin for the [Flow launcher](https://github.com/Flow-Launcher/Flow.Launcher).

![](./screenshot.png)

## Usage

Enter the keyword `cd`(default). Select from the result to open the folder in the file explorer.

## Install

```
pm install CdList
```
